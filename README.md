# claude-code-output-styles

Claude Codeの出力スタイル（output-style）コレクション。
会話の口調やキャラクターを切り替えて、楽しくコーディングできる。

## スタイル一覧

| スタイル | 説明 |
|---|---|
| `announcer` | スポーツ実況アナウンサー風に作業を臨場感たっぷりに中継 |
| `araragi` | 阿良々木暦（化物語）風の自虐的で独白的、言葉遊びと脱線を交えたスタイル |
| `jojo` | JoJoの奇妙な冒険風の口調・擬音を交えたスタイル |
| `kenshiro` | 北斗神拳伝承者ケンシロウの口調で力強く回答 |
| `knt` | わかりやすい言葉と絵文字、京都・滋賀の柔らかい関西弁で親しみやすく回答 |
| `okarin` | 鳳凰院凶真（岡部倫太郎）風の狂気のマッドサイエンティスト |
| `onee` | オネエ言葉で華麗に回答 |
| `porsha` | SING2のポーシャ・クリスタル風の天真爛漫でわがままな口調 |

## セットアップ

`output-styles/` 内の使いたいスタイルの `.md` ファイルを、以下のいずれかにコピーする。

- `~/.claude/output-styles/` — 全プロジェクト共通で使う場合
- `<プロジェクト>/.claude/output-styles/` — 特定のプロジェクトだけで使う場合

## 使い方

Claude Codeで以下のコマンドを実行する。

```
# スタイルを切り替える
/output-style araragi

# 利用可能なスタイルを確認
/output-style
```
